# 📸 Guide des Images du Portfolio

## Structure des images requises

Placez vos images de projets dans ce dossier avec les noms suivants :

### ULM WORLD (3 images)
- `ulm-1.jpg` - Interview Cover
- `ulm-2.jpg` - Promo Post  
- `ulm-3.jpg` - Collabs

### ICC Association (2 images)
- `icc-1.jpg` - Formation
- `icc-2.jpg` - Événementiel

### Restaurant & Food (2 images)
- `bangou-1.jpg` - Menu Digital
- `bangou-2.jpg` - Social Media

### Livres & Auteurs (1 image)
- `livre-1.jpg` - Du Chaos À L'Ordre Divin

## ✨ Spécifications recommandées

- **Format** : JPG ou PNG
- **Dimensions** : 800x600px minimum (idéalement 1200x900px)
- **Taille** : 50-200 KB par image
- **Format** : Landscape (paysage)

## 🚀 Ajouter vos images

1. Téléchargez ou placez vos images dans ce dossier
2. Renommez-les selon les noms ci-dessus
3. Rafraîchissez votre site - les images s'afficheront automatiquement

**Note** : Si une image est manquante, un placeholder générique s'affichera automatiquement.

## 📝 Description des projets

Le portfolio contient 12 projets pré-configurés que vous pouvez personnaliser dans le fichier `inddexs.html` (section "Projects Data").

Chaque projet inclut :
- Titre et catégorie
- Tags/mots-clés
- Description courte
- Liste des livrables

Modifiez l'array `projects` dans le script pour ajouter ou modifier des projets.
